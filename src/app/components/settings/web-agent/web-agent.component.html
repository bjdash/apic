APIC Web Agent

The web version of APIC suffers from the limitation of CORS. To bypass the CORS limitation you can use APIC's web agent to make API calls. Using the web agent will allow you to make the request via the agent and then show the response in the web app. This way you can bypass the CORS limitation set by browsers. To learn more please visit our APIC Agent docs page.

Alternatively you can use APIC's CORS extension for Chrome/Edge to bypass the CORS limitation. We recomend using the native apps or browser extenstion which doesn't have any cors limitation.
